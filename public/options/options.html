Better Search Settings

Configure the keyboard shortcut used to toggle the Better Search overlay.

Shortcut

Press the new key combination inside the capture box, then save. The shortcut must include at least one modifier key.

Default shortcut is Command + Shift + F on macOS, Ctrl + Shift + F elsewhere.

Need a different shortcut?

If the chosen combination is not supported, Chrome may require you to update it from chrome://extensions/shortcuts .
